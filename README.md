# Portfolio Website

A modern front-end portfolio website created to showcase projects, web development skills, and creative UI design.  
This website reflects a professional developer portfolio with a dark aurora-inspired theme, responsive layout, and interactive user experience.

## Technologies Used

- **HTML5** — for semantic structure and page content
- **CSS3** — for custom styling, animations, and layout
- **JavaScript (ES6+)** — for interactivity and dynamic functionality
- **Tailwind CSS** — for responsive utility-first styling
- **Font Awesome** — for icons and UI elements
- **Custom Tailwind Configuration** — for extended colors and animations

## Setup Instructions

To run this project locally:

1. **Clone or download the repository**


2. **Open the project folder**
   Make sure all project files are inside the same directory.

3. **Check the main files**
   The project should include files such as:
   - `index.html`
   - `about.html`
   - `projects.html`
   - `contact.html`
   - `style.css`
   - `script.js`
   - `tailwind-config.js`

4. **Run the website**
   Open `index.html` in your browser.

5. **Optional**
   For better local testing, run the project using a local server such as:
   - VS Code Live Server
   - XAMPP
   - Python HTTP server
